This plugin allows you to bulk copy your menus between sites on your network which are using the same theme. 

== Description ==
Have you ever wanted to duplicate your menus across multiple websites on your wordpress network, and decided that it was too hard to do manually? 

This is what this plugin helps you exactly achieve in a matter of clicks.


== Installation ==
1. Upload your plugin manually or install it via wordpress plugins area
1. Go to your network admin area
1. Go to Settings->Menu copier
1. Follow the instructions on that page
